# 3.2.4.2.7. Interactive content

- インタラクティブコンテンツは、特にユーザーとの対話を目的としたコンテンツです。
- tabindex属性は、任意の要素をインタラクティブコンテンツにすることもできます。

- `<a>`
    - `href`属性がある場合
- `<audio>`
    - `controls`属性がある場合
- `<button>`
- `<details>`
- `<embed>`
- `<iframe>`
- `<img>`
    - `usemap`属性がある場合
- `<input>`
    - `type`属性がhiddenでは無い場合
- `<label>`
- `<select>`
- `<textarea>`
- `<video>`
    - `controls`属性がhiddenでは無い場合
